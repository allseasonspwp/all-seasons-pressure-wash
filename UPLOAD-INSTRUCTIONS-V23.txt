ALL SEASONS WEBSITE — VERSION 23

NEW QUOTE FORM FEATURES
- Customers can select multiple services.
- Customers can add up to 5 project photos.
- Accepted image types: JPG, PNG, WEBP, HEIC and HEIF.
- Maximum size: 5 MB per photo.
- Added client-side validation and mobile-friendly styling.

IMPORTANT WEB3FORMS REQUIREMENT
Multiple photo uploads use Web3Forms Advanced File Uploader, which requires an active paid Web3Forms plan.
Without the paid feature, the rest of the quote form will still be present, but photo attachments may not be delivered.
